Quotient Server Tools APT Repository

This is an APT repository for Quotient server automation tools.

Setup Instructions

Run the bootstrap script on your Ubuntu server:

curl -fsSL https://raw.githubusercontent.com/quotient-research/quotient-server-tools/main/hardware/package/bootstrap-repo.sh | sudo bash

Or manually add the repository:

curl -fsSL https://quotient-research.github.io/quotient-server-tools/apt-repo/repo.gpg | sudo gpg --dearmor -o /usr/share/keyrings/quotient.gpg
echo "deb [signed-by=/usr/share/keyrings/quotient.gpg] https://quotient-research.github.io/quotient-server-tools/apt-repo stable main" | sudo tee /etc/apt/sources.list.d/quotient.list
sudo apt-get update
sudo apt-get install quotient-server-tools
